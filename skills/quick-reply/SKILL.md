# Quick Reply

## Overview

Generates contextually appropriate replies to emails and messages, matching the desired tone and addressing all points raised.

## When to Use

- User pastes a message and asks for a reply
- User needs to respond professionally to an email
- User wants to decline, accept, or follow up on something

## Reply Styles

### Professional
```
Hi [Name],

Thank you for reaching out. [Direct response to main point].

[Additional details if needed]

Best regards,
[Name]
```

### Friendly
```
Hey [Name]!

[Warm response]. [Answer/Comment].

[Casual sign-off]
```

### Brief
```
[Name] - [One-line response]. [Action if needed].
```

### Declining Politely
```
Hi [Name],

Thank you for thinking of me for [opportunity]. Unfortunately, I won't be able to [participate/attend/help] due to [brief reason].

[Optional alternative suggestion]

Best,
[Name]
```

### Following Up
```
Hi [Name],

I wanted to follow up on [topic] from [timeframe]. [Specific question or request].

Please let me know [what you need].

Thanks,
[Name]
```

## Guidelines

1. **Match the tone** of the original message
2. **Address all points** raised in the original
3. **Be concise** - respect the reader's time
4. **Clear next steps** - what action is needed
5. **Offer alternatives** when declining

## Ask If Unclear

- "What tone do you want? (professional/friendly/brief)"
- "Should I accept, decline, or ask for more info?"
- "Any specific points you want to emphasize?"
