## 🎯 Two Ways to Use Zenthral

### ☁️ Cloud Mode (Recommended for Beginners)
**No installation required!** Everything runs in your browser.

- ✅ No software to install
- ✅ Works on any device (phone, tablet, computer)
- ✅ API keys stored securely in the cloud
- ✅ Perfect for simple workflows
- ⚠️ Limited to 5-minute workflows

### 💻 Local Mode (Advanced Users)
Requires installing desktop app. More powerful.

- ✅ Unlimited workflow complexity
- ✅ API keys stay on your computer
- ✅ Access to all 17 agents

## ☁️ Cloud Mode Setup (30 Seconds)

### Step 1: Sign Up
1. Go to your Zenthral URL (e.g., `https://my-zenthral.lovable.app`)
2. Click **"Sign Up"**
3. Enter your email and password
4. Verify your email

### Step 2: Add Your AI Key
1. Click **"Settings"** → **"AI Keys"**
2. You need ONE of these (pick the cheapest):
   - **OpenAI** ($5 credit for new users): https://platform.openai.com/api-keys
   - **Anthropic** ($5 credit for new users): https://console.anthropic.com/
   - **Google Gemini** (Free tier): https://aistudio.google.com/apikey

3. Copy your API key
4. Paste it in Zenthral
5. Click **"Test & Save"**
6. ✅ Done!

### Step 3: Install Your First Skill
1. Go to **"Marketplace"**
2. Browse skills (e.g., "Invoice Generator", "Email Writer")
3. Click **"Install"**
4. Click **"Run"**
5. Enter your details
6. Get instant results!

### That's It! 🎉
No CLI, no daemon, no Python. Just works!

## ❓ FAQ

### Do I need to know how to code?
**No!** Cloud mode is 100% no-code. Just fill in forms and click buttons.

### What if I don't have an AI API key?
You need ONE key from OpenAI, Anthropic, or Google. They all offer $5 free credit for new users.

### Can I try it without paying?
Yes! Use the free credits from OpenAI/Anthropic. That's 500+ workflows for free.

### What if I want more power later?
You can switch to Local Mode anytime by downloading the desktop app.

### Where is my data stored?
- Workflows: Supabase (encrypted PostgreSQL database)
- API keys: Supabase (AES-256 encrypted)
- Execution logs: Deleted after 30 days

### Can I delete my account?
Yes! Go to Settings → Delete Account. All your data is permanently deleted.
