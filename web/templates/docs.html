Introduction

Zenthral is a powerful AI platform that provides pre-built agents, customizable skills, and flexible automation capabilities. Built on top of leading AI models like Claude and GPT, Zenthral makes it easy to integrate AI into your workflows.

Key Features

  • 17 Pre-built Agents - Ready-to-use AI agents for common tasks
  • Custom Skills - Extend functionality with custom Python skills
  • Multiple Channels - CLI, Web, API, and more
  • Session Management - Track conversations and usage
  • Automation - Schedule recurring tasks with cron jobs
  • Multi-Model Support - Use Claude, GPT, or your own API keys
💡
Tip: Start with the Quick Start guide to get up and running in under 5 minutes.

Quick Start

Get Zenthral running in just a few steps:

1. Installation

git clone https://github.com/yourusername/zenthral.git
cd zenthral
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

2. Configure API Keys

Set up your AI provider credentials:

export ANTHROPIC_API_KEY=your-key-here
export OPENAI_API_KEY=your-key-here

3. Start the Web Dashboard

cd web
python3 app.py

Visit http://localhost:5001 to access the dashboard.

✅
Success! You're now ready to start using Zenthral.

API Setup

Zenthral supports multiple AI providers. You can use our managed API (pay-as-you-go) or bring your own API keys.

Managed API (Recommended)

  • No API keys required
  • Pay only for what you use
  • Automatic failover and redundancy
  • Volume discounts available

Custom API Keys

To use your own API keys:

  1. Navigate to Dashboard → AI Source
  2. Select Custom API
  3. Enter your Anthropic and/or OpenAI API keys
  4. Click Continue to Setup
⚠️
Important: API keys are stored in your browser session and never sent to our servers.
